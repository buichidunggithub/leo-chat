# leo-chat

# Create database
Create mongodb on https://cloud.mongodb.com/

Create .env in folder "server" by the command touch .env <br/>
.env file includes: <br/>
PORT= <br/>
MONGO_URL= <br/>

# Start client
cd client <br/>
npm i --save <br/>
npm start <br/>

# Start server
cd server <br/>
npm i --save <br/>
npm start <br/>
